# IDS_NDL_Data_set

About NSL-KDD Dataset The NSL-KDD dataset is a benchmark dataset for network intrusion detection. It contains network traffic data labeled as either 'normal' or various types of 'attacks'. Dataset Features (41 total) •	Basic features: duration, protocol_type, service, flag •	Content features: src_bytes, dst_bytes, wrong_fragment, urgent, hot, etc. •	Traffic features: count, srv_count, error rates, same_srv_rate, etc. •	Host-based features: dst_host_count, dst_host_srv_count, etc.
